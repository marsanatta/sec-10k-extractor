# Prompts

This folder records the key prompts used to build this project, as required by the
assignment ([../ASSIGNMENT.md](../ASSIGNMENT.md)). Reviewers read these to understand the
AI-collaboration process.

## Format

Each prompt is saved as one markdown file by the `/save-prompt` skill, named:

```
YYYY-MM-DD-<short-summary-slug>.md
```

Each file contains:

- The verbatim prompt that was given
- A short summary
- A high-level record of the resulting action / response

Files are ordered chronologically by the date prefix.
